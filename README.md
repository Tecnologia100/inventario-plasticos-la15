# 🏭 Plásticos La 15 — Sistema de Inventario

Sistema de control de inventario web para Plásticos La 15.  
**Cero instalaciones** — funciona directo en el navegador.

## 🚀 Cómo usar

### Opción 1: GitHub Pages (recomendado)
1. Sube este repositorio a GitHub
2. Ve a **Settings → Pages → Branch: main → Save**
3. Accede desde cualquier dispositivo en: `https://tu-usuario.github.io/tu-repo/`

### Opción 2: Abrir local
1. Abre `index.html` directamente en el navegador (Chrome recomendado)

## 📱 Acceso desde el celular
Usa la misma URL de GitHub Pages — funciona en cualquier navegador móvil.

## 💾 Backups
Los datos se guardan en el **localStorage** del navegador.  
Usa el botón **"Descargar Backup"** periódicamente para respaldar.  
Si cambias de navegador o PC, usa **"Restaurar"** para cargar el backup.

## 📊 Funcionalidades
- 160 productos importados del catálogo Excel
- Entradas y salidas de inventario con validación
- Semáforo de alertas (🔴🟡🟢🟠⚪)
- Historial de movimientos filtrable
- Diseño responsive (PC + móvil)
- Backup/restauración de datos en JSON
